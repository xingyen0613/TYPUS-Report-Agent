# Typus Data 模板

## 说明

此文件展示 Typus Data 的数据格式。请根据此格式准备您的实际数据。

**重要**：系统会忽略 Column C (TVL_Total)，使用新公式自动计算。

---

## TVL 数据

| Date | Month | TVL (忽略) | Perps TVL | DOV TVL | SAFU TVL | Calculated Total TVL | Calculated Options TVL |
|------|-------|-----------|-----------|---------|----------|---------------------|----------------------|
| 2025-01-01 | Jan | (忽略) | $5,200,000 | $10,500,000 | $1,800,000 | $19,300,000 | $13,100,000 |
| 2025-01-31 | Jan | (忽略) | $5,500,000 | $11,200,000 | $1,900,000 | $20,500,000 | $14,000,000 |
| 2024-12-01 | Dec | (忽略) | $4,800,000 | $11,800,000 | $2,100,000 | $21,100,000 | $16,000,000 |
| 2024-12-31 | Dec | (忽略) | $5,100,000 | $10,800,000 | $1,900,000 | $20,600,000 | $14,600,000 |

### 计算公式

- **Total TVL** = Perps TVL + DOV TVL + (2 × SAFU TVL)
  - 例如：$5,200,000 + $10,500,000 + (2 × $1,800,000) = $19,300,000

- **Options TVL** = DOV TVL + (2 × SAFU TVL)
  - 例如：$10,500,000 + (2 × $1,800,000) = $13,100,000

## Users 数据

| Date | Month | MAU - Typus Perp | MAU - Typus v2 | Total MAU | Total Users (GA) |
|------|-------|-----------------|----------------|-----------|------------------|
| 2025-01-31 | Jan | 1,250 | 600 | 1,850 | 3,200 |
| 2024-12-31 | Dec | 1,400 | 700 | 2,100 | 3,500 |

### 定义

- **Core MAU** = MAU - Typus Perp + MAU - Typus v2
  - 例如：1,250 + 600 = 1,850

- **Total Users (GA)** = 与核心产品互动的总独立用户数
  - 描述为："total unique users interacting with our core product sections"

## 数据要求

### 必需数据

为了生成月报，**必须**包含：

- [x] 当月 (M) 的 TVL 数据（月初 + 月末）
- [x] 上月 (M-1) 的 TVL 数据（月初 + 月末）
- [x] 当月 (M) 的 Users 数据（月末）
- [x] 上月 (M-1) 的 Users 数据（月末）

### 推荐数据

建议额外包含：

- [ ] Perps Volume 和 TLP Fees
- [ ] Options Volume 和 Contracts
- [ ] 其他产品指标

---

## 如何使用

1. **复制此模板**
   ```bash
   cp TEMPLATE.md typus-data.md
   ```

2. **填入实际数据**
   - 更新 TVL 数据（Perps, DOV, SAFU）
   - 更新 Users 数据（MAU）
   - 更新其他指标（可选）

3. **验证数据**
   ```bash
   /monthly-report-prepare
   ```

4. **如果验证通过，继续下一步**
   ```bash
   /fetch-market-prices
   ```

---

## 数据来源

- **原始数据**：从 Google Sheets "Typus Data" 导出
- **更新频率**：每月更新一次（月末）
- **数据格式**：Markdown 表格或 CSV

---

## 注意事项

- **忽略 Column C**：系统会自动使用新公式计算 Total TVL
- **月增长率**：系统会自动计算 (M vs M-1)
- **数据简化**：系统会自动格式化（$19.1M, $111k, +29%）
- **必需 M-1 数据**：没有上月数据无法计算增长率

---

**准备好数据后，运行 `/monthly-report-prepare` 验证数据完整性。**
